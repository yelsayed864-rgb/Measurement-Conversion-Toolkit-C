# Measurement Conversion Toolkit (C)

A console-based C program that converts between different unit systems such as length and temperature.

## Features
- Meters to Feet conversion
- Celsius to Fahrenheit conversion
- User input handling
- Clear prompts and error handling

## Concepts Used
- Conditional statements
- Arithmetic operations
- User input/output
- Basic program flow control

## How to Run

Compile:
gcc main.c -o converter


Run:


./converter
